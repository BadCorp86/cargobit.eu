# 4. Level Definitions

## L1 — Junior Engineer

### Responsibilities
- Learns codebase and tools
- Completes assigned tasks
- Asks questions and seeks feedback
- Writes tests for own code

### Expectations (6-12 months)
- Consistently delivers quality code
- Understands team processes
- Participates in code reviews
- Demonstrates growth mindset

---

## L2 — Engineer

### Responsibilities
- Delivers features independently
- Writes maintainable, tested code
- Participates in on-call rotation
- Mentors junior engineers

### Expectations
- Owns features end-to-end
- Identifies and fixes issues proactively
- Contributes to documentation
- Collaborates effectively with team

---

## L3 — Senior Engineer

### Responsibilities
- Leads complex projects
- Designs system components
- Conducts thorough code reviews
- Mentors multiple engineers
- Drives technical decisions

### Expectations
- Delivers high-impact projects
- Improves team processes
- Reduces technical debt
- Recognized as technical expert

---

## L4 — Staff Engineer

### Responsibilities
- Defines architecture patterns
- Leads cross-team initiatives
- Influences technical strategy
- Resolves complex technical conflicts
- Mentors senior engineers

### Expectations
- Solves organization-level problems
- Creates reusable solutions
- Improves engineering practices
- Represents engineering to leadership

---

## L5 — Principal Engineer

### Responsibilities
- Sets technical vision
- Drives strategic initiatives
- Influences company direction
- Builds engineering culture
- External representation

### Expectations
- Shapes company strategy
- Creates lasting technical value
- Develops engineering leaders
- Industry recognition

# 5. Promotion Criteria

## 5.1 Process

1. Self-nomination or manager nomination
2. Evidence collection (impact, competencies)
3. Peer feedback
4. Calibration session
5. Decision and communication

## 5.2 Requirements

| From | To | Key Requirement |
|------|----|-----------------|
| L1 | L2 | Consistent independent delivery |
| L2 | L3 | Project leadership + mentoring |
| L3 | L4 | Cross-team impact + architecture |
| L4 | L5 | Organization-level strategy |
