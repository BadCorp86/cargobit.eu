# CargoBit System Glossary (Extended)
Version 1.0
Internal Use Only

---

# 1. Purpose

Dieses Dokument enthält ein erweitertes Glossar aller wichtigen Begriffe im CargoBit System.

---

# 2. A

| Term | Definition |
|------|------------|
| ADR | Architecture Decision Record - Dokumentation einer Architekturentscheidung |
| API | Application Programming Interface - Programmierschnittstelle |
| Audit Log | Unveränderliches Protokoll aller Systemereignisse |
| Availability | Verfügbarkeit des Systems, gemessen als Prozentsatz der Betriebszeit |
| Auto-scaling | Automatische Anpassung der Ressourcen basierend auf Last |

---

# 3. B

| Term | Definition |
|------|------------|
| Backoff | Verzögerungsstrategie bei Fehlversuchen (exponentiell) |
| Backup | Kopie der Daten für Wiederherstellungszwecke |
| Balance | Kontostand in einer Wallet |
| BCP | Business Continuity Plan - Plan zur Geschäftskontinuität |
| Bearer Token | Authentifizierungstoken für API-Zugriff |

---

# 4. C

| Term | Definition |
|------|------------|
| Caching | Zwischenspeicherung von Daten für schnelleren Zugriff |
| Checksum | Prüfsumme zur Verifizierung der Datenintegrität |
| CI/CD | Continuous Integration/Continuous Deployment |
| Compliance | Einhaltung von Vorschriften und Standards |
| Correlation ID | Eindeutige Kennung zur Nachverfolgung von Requests |
| Credit | Gutschrift auf einer Wallet |
| CronJob | Zeitgesteuerter automatischer Task |

---

# 5. D

| Term | Definition |
|------|------------|
| Data Classification | Einstufung von Daten nach Sensibilität |

| Determinism | Determinismus - gleiche Eingabe ergibt gleiche Ausgabe |
| DPA | Data Processing Agreement - Datenverarbeitungsvertrag |
| DPO | Data Protection Officer - Datenschutzbeauftragter |
| DRP | Disaster Recovery Plan - Katastrophenwiederherstellungsplan |

---

# 6. E

| Term | Definition |
|------|------------|
| Encryption | Verschlüsselung von Daten |

| Event | Zustandsänderung im System (z.B. payment.succeeded) |
| Exponential Backoff | Exponentiell wachsende Wartezeit zwischen Wiederholungen |

| Term | Definition |
|------|------------|
| Failover | Automatischer Wechsel zu einem Reservesystem |
