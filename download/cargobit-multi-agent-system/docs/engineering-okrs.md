Version 1.0
Internal Use Only

# 1. Purpose

Dieses Dokument definiert die Objectives and Key Results (OKRs) für das Engineering-Team bei CargoBit. Es stellt sicher, dass Ziele klar, messbar und ambitioniert sind.

---

# 2. OKR Framework

| Component | Description |
|-----------|-------------|
| Objective | What we want to achieve |
| Key Result | How we measure success |
| Initiative | What we do to achieve it |

# 4. OKR Process

## 4.1 Quarterly Cycle

| Week | Activity |
|------|----------|
| Week 1 | Set OKRs for quarter |
| Week 2-12 | Execute and track |
| Week 13 | Review and score |

## 4.2 Scoring

| Score | Description |
|-------|-------------|
| 0.0-0.3 | Failed to achieve |
| 0.4-0.6 | Made progress |
| 0.7-0.9 | Nearly achieved |
| 1.0 | Fully achieved |

## 4.3 Check-ins

- Weekly: Progress update in team meeting
- Bi-weekly: Detailed review with manager
- Monthly: Leadership review
